# Smart Retail Warehouse Management System

## Overview
An IoT-based warehouse monitoring system developed using ESP32, RFID, Load Cell and Firebase.

## Features
- Real-time stock monitoring
- RFID inventory tracking
- Theft detection
- Low stock alerts
- Firebase integration

## Technologies Used
- ESP32
- RFID RC522
- HX711 Load Cell
- Firebase
- MIT App Inventor

## My Contribution
Designed and integrated the complete hardware setup, ESP32 firmware, sensor interfacing, and Firebase communication.
